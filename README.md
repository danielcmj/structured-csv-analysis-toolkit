\# Sensor Data Analysis Pipeline (Python)



This repository contains a reusable Python pipeline for organizing, loading, and visualizing experimental measurement files.



\## What's included

\- Python code for metadata extraction, dataset loading, filtering/grouping, and plotting

\- A short PDF guide describing the expected folder structure and available functions



\## Data confidentiality

Raw measurement data is not included in this repository.



\## Quick start

1\. Create a virtual environment

2\. Install dependencies from `requirements.txt`

3\. Place your measurement folders locally (not tracked by git)

4\. Run the analysis script from `src/`
